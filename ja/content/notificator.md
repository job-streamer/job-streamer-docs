# Notificator
Notificator は Control-bus から命令を受け取り、任意の API を呼び出すサーバです。
[Apache Camel](http://camel.apache.org/) のラッパークラスとして実装されており、仕様は Apache Camel に基づきます。

## Notificator でできること
Notificator を用いると「あるジョブが終わったときメールを送信したい」などの要件を満たすことが可能となります。
詳細は [通知サーバの設定](./notificator-config.html) を参照してください。
その他に何が出来るかについては [Apache Camel](http://camel.apache.org/) を参照して下さい。
